# 📌 Overview

AMIT HOSPITAL is a premium healthcare web platform designed specifically for a specialized hospital focusing on:

- Pulmonary Care  
- Respiratory Disorders  
- Advanced Chest Diagnostics  
- Critical Respiratory Monitoring  
- Patient-Centered Healthcare Experience  

The project focuses heavily on **premium design language**, creating a healthcare experience that feels modern, trustworthy, elegant, and technologically advanced.

---

# 🎨 Design Philosophy

The website design is inspired by premium digital experiences rather than traditional healthcare templates.

Core principles:

- Luxury visual design  
- Glassmorphism based UI  
- Smooth animations and transitions  
- Minimalistic premium aesthetic  
- Trust-oriented healthcare branding  
- High-end client presentation quality  

Design inspiration references:

- Apple style visual language  
- Enterprise healthcare interfaces  
- Premium Webflow design systems

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/amit-hospital.git
```

Move into project directory

```bash
cd amit-hospital
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
